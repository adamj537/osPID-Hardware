Readme
******
1. This repository contains all the hardware design files of osPID which includes the most recent version and 
older revision.
2. You will need Diptrace v2.2 & above to open the schematic & PCB design files.
Free version of Diptrace is available for download here: 
http://www.diptrace.com/downloads/dipfree_en.exe
3. The component & footprint libraries used can be found here:
https://github.com/rocketscream/Rocket-Scream-Diptrace-Libraries
